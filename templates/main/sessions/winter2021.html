Winter 2021 Session

Curriculum

As members of the Millburn Robotics Program (winners of the VEX Robotics 2018-2019 World Championship as well as recipients of numerous awards), we have accumulated a lot of experience in the fields of programming and engineering. However, due to Covid-19, our ability to have students physically handle robots and learn engineering concepts through practice has been greatly hampered. Nonetheless, an online model that focuses on programming is still possible, and we plan to take the opportunity presented by winter break to teach students core concepts in robotics programming using the VEXcode VR robot simulation software. Lessons will touch upon concepts including but not limited to:

  • Block code instruction, and depending on degree of individual success with basic concepts, written code instruction using Python
  • Using programming structures such as loops and conditionals to optimize code
  • Precision functions such as proportional slowdown

Logistics

  • The Essex IQ Winter Session is a one week program that will be taking place from 12/28/2020 - 01/01/2021
  • All meetings will take place over Zoom call in observance of social distancing guidelines
  • Meetings will be 2 hours
  • Just sign up and attend -- no other barriers to entry!
